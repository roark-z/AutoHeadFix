AHF\_Datalogger 
=======================

The datalogger implemented in the code is determined through the jsn file for the relevant task. Currently, :code:`AHF_DataLogger_textMySql` is used for all tasks; however, SQL logging is currently disabled due to compatibility issues.

Note: hdf5 logs are created independent of this module, in :code:`AHF_Stimulus_Laser.py`
